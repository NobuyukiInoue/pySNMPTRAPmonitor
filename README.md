# pySNMPTRAPmonitor

Monitor and Dump SNMP TRAP by Python3


## Requirements

Python 3.5 or later.


### Supported OS

* MS-Windows
* macOS
* Linux


## How to execute pySNMPTRAPmonitor

```
python snmptrapm.py [timeout(s)] [testmode]
```
